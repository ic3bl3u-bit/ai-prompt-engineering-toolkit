# Contributing to AI Prompt Engineering Toolkit

Thanks for your interest in contributing! Quality prompts help everyone.

## How to Contribute a Prompt

1. Fork this repo
2. Add your prompt to `prompts/[category].md` (create the file if it doesn't exist)
3. Follow the prompt template below
4. Submit a PR with the title: `Add prompt: [brief description]`

## Prompt Template

```markdown
### [Prompt Name]

**Category:** [marketing/sales/dev/productivity/research/writing]
**Best for:** [specific use case]

\```
[The actual prompt — following the RTFC framework: Role, Task, Format, Constraints]
\```

**Why it works:** [1-2 sentences explaining the prompt design]
```

## Prompt Quality Standards

- Must follow the **RTFC framework** (Role, Task, Format, Constraints)
- Must include `[bracketed]` placeholders for user-specific details
- Must be tested in at least one AI tool (ChatGPT, Claude, or Gemini)
- Must produce specific, structured output (not vague responses)
- Must be original (not copied from other prompt collections)

## Categories

| Category | Description |
|----------|-------------|
| `marketing` | Content strategy, email sequences, social media, SEO |
| `sales` | Cold emails, follow-ups, proposals, negotiation |
| `development` | Code review, debugging, architecture, documentation |
| `productivity` | Meeting notes, project management, planning, automation |
| `research` | Data analysis, competitor intelligence, synthesis |
| `writing` | Blog posts, copywriting, editing, storytelling |

## Code of Conduct

Be respectful. No spam. No promotional content in prompts (product links are in the README and category prompt files).
