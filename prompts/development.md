## Prompt Framework

Every prompt follows the **RTFC Framework**:

1. **R**ole — "You are a senior software engineer"
2. **T**ask — "Review this code for bugs and performance"
3. **F**ormat — "Categorize: Critical, Improvements, Suggestions"
4. **C**onstraints — "Include line numbers and corrected code"
